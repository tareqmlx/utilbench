The system, but sticky.
Workbench, v1.0

A loud, friendly system for a developer toolbox: warm cream paper, ink black outlines, hard pop-shadows, and a fixed pastel sticker palette. Use this page as the source of truth when extending the app.

§ 01 FOUNDATIONS

What's set in glue

Principles

  • Sticker, not shadow. Borders are 2px solid ink. Shadows are hard offsets — never blurred, never opacity'd.
  • Paper, not gradient. One warm cream background. Polka-dot pattern lives at the page level, not inside cards.
  • Pastels, fixed set. Pink, lilac, mint, sky, lemon — five flavors only. Tomato & grass are reserved for CTA / status.
  • Type with a smile. Bricolage Grotesque does the loud bits, Inter handles body, Geist Mono runs the meta.
  • Tilt, but rarely. Stickers in the hero may rotate ±4–6°. Cards stay square. Don't tilt the world.
